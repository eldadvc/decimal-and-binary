Decimal to Binary Easy Peasy

What you need to know:
  • Binary numbers are numbers just like any other number.
  • You can count anything using binary numbers, just like you would using decimal numbers.
  • While decimal numbers consist of ten digits (0 - 9, representing ten values), binary numbers consist of two digits (0 - 1, representing two values).
  • With decimal numbers, we add a 'position' to the left of the number containing another digit to represent a quantity larger than 9. For a number larger than 99, we add another position.
  • All decimal positions are powers of 10 (1, 10, 100, 1000, 10000, and so on).
  • We do the same for binary numbers, the only difference being (see next bullet).
  • All binary number positions are powers of 2.
  • The positions in a binary number consisting of 8 digits are (from left to right: 128, 64, 32, 16, 8, 4, 2, 1)
  • Therefore the binary number 11000001 converted to decimal is 193.
  • 1x128 + 1x64 + 1x0 + 1x0 + 1x0 + 1x0 + 1x0 +1x1 = 128 + 64 + 0 + 0 + 0 + 0 + 0 +1 = 193.
  • Try it yourself; practice makes perfect.

How this Works

  • The container to the right displaying 0 will show the value of the binary number you construct using the row of eight ones and zeros in decimal. Hovering over and clicking them will show the value of the binary position.
  • Clicking on the New Number button will generate a decimal number. Your task is to use the row of eight ones and zeros to create the binary equivalent.
